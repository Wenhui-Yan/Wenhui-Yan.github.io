A One-Page Reading Sheet

2025-12-30 · Notes

I started using a one-page format for reading notes: one-sentence summary, three key insights, and one action item.

One-Sentence Summary

Make sure I can recall the core logic of the entire book within 30 seconds.

Key Insights

Keep only ideas that can change actions or decisions.

Action Item

Each book should lead to one small real-world experiment.
